Turn calculator

For submerged turn calculating. Start by setting in the initial course and new course and then start the turn with full rudder deflection (35 degrees). Once the turn has begun, reset the odometer by using the button below the speedometer that is above the navigation chart or by pressing Z by default while in the navigation chart and place a marker on your current position on the map. When the turn is completed, reset the odometer once again, set the value from the odometer to the field below and press "Calculate"
